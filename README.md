## Apache Airflow running in Astronomer

This is a basic Apache Airflow initialization project in Astronomer.

## Requirements

- Python 3.x 
- Astronomer
- Docker 

## Execution

1. Run the following command to execute the project in the local environment

```bash
astro dev start
```
